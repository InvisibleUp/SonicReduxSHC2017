HOW TO USE

To edit character models:
    - Open "exportme.blend" in Blender.
    - Make required edits
    - Export to OBJ
    - Use Python 3 on the commandline to run obj2srm.py
    - Copy resulting .bin file to bin/chars/#/model.srm
    
To edit character attributes
    - Open "attribs.dat" in a spreadhseet or text editor program
    - Make required edits. Save as CSV if prompted.
    - Use Python 3 on the commandline to run makeattribs.py
    - Copy resulting .dat file to bin/chars/#/attribs.dat
    
See https://github.com/InvisibleUp/Sonic-R-Utilities for more details, as most of these utilities are from there.
